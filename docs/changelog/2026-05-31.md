# 更新日志 — 2026-05-31

记录 2026-05-31 的变更，供发布说明与回溯使用。

主要变更
- 新增静默 Windows 启动器：`run_clicker.vbs`
  - 行为说明：优先尝试在发布目录下启动 `RocoKingdom_Clicker.exe --gui`；若不存在则回退到本地虚拟环境的 `pythonw` 启动 `Clicker.py --gui`，并会使用 ShellExecute 的 `runas` 请求 UAC 提升（便于在游戏进程中捕获热键）。
- GUI 响应性改进：修复脚本列表选择延迟，`webview.py` 已绑定 Listbox 的 `<<ListboxSelect>>` 事件，点击脚本时左侧的“当前选择脚本”会立即刷新。
- 脚本引擎增强：新增 `timed` 动作以支持脚本内的定时关闭或条件停止；调整若干脚本默认参数以提高类人化效果（更合理的 `hold_ms`、jitter 支持）。
- 全局行为调整：`move_mouse` 现为全局开关，ActionExecutor 会统计并在 GUI 中展示 `ignored_moves`（因禁用移动而跳过移动的次数）。

变更影响的文件（摘要）
- `webview.py`：绑定列表选择事件；“当前选择脚本”仅显示脚本名，运行/暂停/倒计时统一显示在状态栏。
- `Clicker.py`：暴露脚本会话状态（`current_script_name`, `script_running`, `script_paused`），并在会话生命周期中维护这些状态；添加热键监听日志以便调试 F1/F2 等热键行为。
- `ActionScript.py`：新增 `TimedAction` 支持；ActionExecutor 接受全局 `move_mouse` 设置并统计 `ignored_moves`。
- `run_clicker.vbs`：新增启动器，采用 ShellExecute `runas` 发起 UAC 提升（供在需要管理员权限时使用）。
- `build_release.bat`：复制 VBS、使用 `--windowed` 打包并生成 release 压缩包。

快速验证步骤
1. 双击 `run_clicker.vbs`：应弹出 UAC 提示（若需要），确认后打开 GUI，且没有独立的命令行窗口留在前台。
2. 在 GUI 列表中点击任意脚本：左侧“当前选择脚本”标签应立即更新且仅显示脚本名；运行/暂停/倒计时在“状态”栏显示。
3. 运行 `build_release.bat` 并检查 `dist\\RocoKingdom_Clicker`：应包含 `run_clicker.vbs`、`run_clicker.bat`、示例脚本与默认配置文件。

调试与注意事项
- 若在游戏内无法响应热键，请确保以管理员权限启动（运行 `run_clicker.vbs` 并接受 UAC 提升）。
- 已添加热键钩子层面的调试日志（hotkey producer / consumer），如遇 F1/F2 触发不稳定，请提供日志以便进一步分析是否需要去重或节流处理。
